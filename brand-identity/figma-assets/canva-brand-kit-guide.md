# Canva Brand Kit Setup Guide

## Setting Up Tachyon Labs Brand Kit in Canva

### 1. Upload Logo Files

In Canva Brand Kit, upload these files:

**Primary Logos**
- `tachyon-labs-logo-horizontal.svg` - Main logo
- `samaritan-logo-horizontal.svg` - Product logo
- `quantum-loop-icon.svg` - Icon only

**Logo Variations**
- `tachyon-labs-logo-vertical.svg` - For square formats
- `samaritan-logo-vertical.svg` - For square formats
- `tachyon-labs-wordmark.svg` - When icon appears separately
- `samaritan-wordmark.svg` - When icon appears separately

### 2. Set Brand Colors

Add these colors to your Brand Kit palette:

**Primary Palette**
- Black: #000000 (Primary text and logos)
- White: #FFFFFF (Inverted logos and text)

**Neutral Palette** (for backgrounds and UI)
- Dark: #0a0a0a
- Medium Dark: #262626
- Medium: #737373
- Light: #e5e5e5
- Very Light: #fafafa

**Accent Color**
- Green: #10B981 (Call-to-action and highlights)

### 3. Configure Brand Fonts

Since Mona Sans isn't available in Canva, use these alternatives:

**Headers**: Montserrat
- Main headings: Bold, 48-72pt
- Subheadings: SemiBold, 24-36pt
- Section headers: Medium, 18-20pt

**Body Text**: Open Sans
- Body copy: Regular, 14-16pt
- Captions: Regular, 12pt
- Emphasis: SemiBold, 14-16pt

### 4. Create Brand Templates

Set up these essential templates:

#### Social Media Posts (1080x1080)
- Logo placement: Top left, 48px from edges
- Use vertical logo variant
- Maintain 80px margin on all sides

#### LinkedIn Banner (1584x396)
- Logo placement: Left side, centered vertically
- Use horizontal logo variant
- Keep important content within 1400x396 safe area

#### Presentation Slides (1920x1080)
- Logo placement: Bottom right, 40px from edges
- Use horizontal logo at 160px width
- Title safe area: 120px from all edges

#### Business Card (3.5"x2")
- Front: Logo centered, contact below
- Back: Quantum loop icon pattern
- Bleed: 0.125" on all sides

### 5. Design Guidelines for Canva

#### Logo Usage Rules

**DO:**
- Maintain minimum clear space (equal to icon height)
- Use on solid color backgrounds
- Scale proportionally
- Use provided color variants only

**DON'T:**
- Add effects (shadows, outlines, gradients)
- Change colors outside brand palette
- Place on busy backgrounds
- Rotate or skew

#### Color Combinations

**High Contrast (Recommended)**
- Black on white
- White on black
- Black on light gray (#f5f5f5)
- White on dark gray (#262626)

**Accent Usage**
- Green on white (CTAs)
- White on green (Buttons)
- Green as highlight only (10% max)

#### Typography Hierarchy

1. **Hero Headlines**
   - Montserrat Bold, 48-72pt
   - Line height: 1.2
   - Letter spacing: -0.02em

2. **Section Titles**
   - Montserrat SemiBold, 24-36pt
   - Line height: 1.3
   - Letter spacing: -0.01em

3. **Body Copy**
   - Open Sans Regular, 14-16pt
   - Line height: 1.6
   - Letter spacing: 0

4. **Captions & Labels**
   - Open Sans Regular, 12pt
   - Line height: 1.5
   - Letter spacing: 0.01em

### 7. Export Settings

When downloading from Canva:

**For Web**
- PNG: RGB, Transparent background
- Size: 2x intended display size
- Quality: Highest

**For Print**
- PDF Print: CMYK, 300 DPI
- Bleed: 3mm (0.125")
- Crop marks: Yes

**For Social Media**
- Use platform presets
- Optimize for each platform
- Test on mobile devices

### 8. Brand Consistency Checklist

Before publishing any design:

- [ ] Logo is correctly placed and sized
- [ ] Only brand colors are used
- [ ] Typography follows hierarchy
- [ ] Adequate white space maintained
- [ ] Clear visual hierarchy
- [ ] Accessible contrast ratios
- [ ] Consistent with brand voice

### 9. Common Mistakes to Avoid

1. **Using drop shadows on logos** - Keep it clean
2. **Mixing too many fonts** - Stick to 2 max
3. **Low contrast text** - Ensure readability
4. **Cluttered layouts** - Embrace white space
5. **Off-brand colors** - Use palette only
6. **Stretched logos** - Always proportional

## Pro Tips for Canva

1. **Save brand combinations** as styles
2. **Create folders** for different asset types
3. **Use grids** for consistent alignment
4. **Duplicate** successful designs as starting points
5. **Preview** on different devices before publishing

Remember: Consistency builds recognition. When in doubt, keep it simple and refer to the brand guidelines.
