## Logo Variations

### Tachyon Labs
- **Full Logo**: Includes the Quantum Loop symbol and "Tachyon Labs" text
- **Mark Only**: Just the Quantum Loop symbol without text
- **Black**: For use on light backgrounds
- **White**: For use on dark backgrounds
- **Green Background**: Logo on brand green (#2A6A43) background

### Samaritan
- **Full Logo**: Includes the Quantum Loop symbol and "Samaritan" text
- **Black**: For use on light backgrounds
- **White**: For use on dark backgrounds

## Usage Guidelines

### Clear Space
Maintain clear space around the logo equal to the height of the logomark (the circular symbol).

### Minimum Size
- Digital: 24px height minimum
- Print: 10mm height minimum

### Color Usage
- Use black logos on light backgrounds (white, light gray)
- Use white logos on dark backgrounds (black, dark colors)
- The green accent (#2A6A43) can be used for special applications

### Don'ts
- Do not stretch, distort, or rotate the logo
- Do not change the colors without approval
- Do not add effects like drop shadows or gradients
- Do not place the logo on busy backgrounds that reduce legibility

## File Formats

- **SVG**: Vector format, ideal for web and scalable applications
- **PNG**: Raster format with transparency, suitable for most digital uses

## Brand Colors

- **Primary Black**: #000000
- **Primary White**: #FFFFFF
- **Neutral 950**: #0a0a0a
- **Brand Green**: #2A6A43 (optional accent)

## Typography

- Primary typeface: System UI / Inter
- Weights: Regular, Semibold, Bold
